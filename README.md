알레데이아는 죽은 진리의 신 에무나의 교리를 숭상하는 자들이 모여 형성된 신정 사회였으며, 진리는 신앙이자 법이자 질서였다.</p>  <p>그녀는 진리를 단순한 계율이 아닌, 이해되어야 할 언어로 여겼다.  
사람들과의 교류를 중시했고, 설득과 해석을 통해 진리를 나누려 했다.</p>  
<p>그녀에게 진리는 강요가 아닌 공개였고 계시는 독점되어서는 안 되는 것이었다.</p>  <p>그러나 진리의 초월자가 죽은 시점에서 진리를 단일한 목소리로 묶어줄 사람은 크게 존재하지 않았다.<br>  
초월자가 남긴 교리는 계속해서 남아 있었으나, 그 해석은 점점 갈라지기 시작했다.</p>

그러나 그녀의 노력은 오히려 「진리의 단일화」로 인식되었다.</p>  <p>내전은 피할 수 없게 되었고,
